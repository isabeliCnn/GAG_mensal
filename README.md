# GAG Mensal

Sistema desktop desenvolvido em Java para gerenciamento de bar e restaurante.

## Funcionalidades

* Cadastro de usuários
* Cadastro de produtos
* Controle de estoque
* Registro de pedidos
* Controle de fichas
* Relatórios

## Tecnologias Utilizadas

* Java 17
* Swing
* PostgreSQL
* Hibernate
* Flyway
* Maven
